How to Use TextFusion Pro

This text animation generator creates professional GSAP animations. Here's how to use it:

Basic Controls

  • Custom Text: Enter the text you want to animate
  • Text Style: Apply bold, italic, or underline formatting

Animation Settings

  • Animation Type: Choose from various animation effects
  • Split Type: Break text into characters, words, or lines
  • Stagger Order: Control how animations sequence
  • Speed: Adjust overall animation duration
  • Stagger Delay: Time between each element's animation
  • Easing: Control animation timing functions
    • Select "Custom" to create your own easing curve using the handles
  • Text Mask: Apply reveal effects to your text

Text Styling

  • Font Size: Change text size
  • Font Family: Select typeface
  • Text Color: Customize color
  • Letter Spacing: Adjust space between characters
  • Font Weight: Control text thickness
  • Text Transform: Apply uppercase, lowercase, etc.
  • Text Shadow: Add glow or shadow effects

Advanced Settings

  • Y/X Offset: Starting position for animations
  • Rotation: Spin elements during animation
  • 3D Transforms: Enable perspective effects
  • Multiple Colors: Apply random colors to text
  • Repeat Count: Number of animation loops
  • Blur Effect: Add motion blur
  • Scale Effect: Size change during animation
  • Particles Effect: Create exploding text effects

Preview Controls

  • Device Options: Test animations at different screen sizes
  • Play Button: Run the animation
  • Timeline Scrubber: Manually control animation progress
  • Play/Pause/Reset: Control animation playback

Exporting Animations

  • Generated JavaScript: Ready-to-use GSAP code
  • Copy Code: Copy to clipboard for use in your projects
  • Export: Download the animation code

About TextFusion Pro

What is TextFusion Pro?

TextFusion Pro is a powerful text animation generator that helps designers and developers create stunning text animations using GSAP (GreenSock Animation Platform). It provides an intuitive interface to design complex animations without writing code, then generates the JavaScript needed to implement them in your projects.

Features

  • 19 different animation types
  • Character, word, and line-based animations
  • Custom easing with visual editor
  • Text masking and reveal effects
  • 3D transformations
  • Timeline scrubbing and precise control
  • Multiple device preview
  • Code generation for easy implementation

Technologies

TextFusion Pro is built with:

  • GSAP (GreenSock Animation Platform)
  • SplitType for text splitting
  • Modern HTML, CSS, and JavaScript

Use Cases

Perfect for:

  • Website headers and hero sections
  • Interactive user interfaces
  • Landing pages
  • Digital presentations
  • Motion graphics
  • Educational content about animation
